# Module 00: Basic Python Exercises

This module is designed to help you build a strong foundation in Python before tackling Data Structures and Algorithms.

We will cover:
1.  **Variables & Arithmetic**: Basic math and storing values.
2.  **Conditionals**: Making decisions (if/else).
3.  **Loops**: Repeating actions (for, while).
4.  **Functions**: Reusing code.
5.  **Basic Lists/Strings**: Simple collections.

## How to use
Open the `.py` files and try to solve the problems in the functions provided.
Run the file to see if your solutions pass the basic tests.
