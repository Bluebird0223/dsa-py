## What is Dynamic Programming?

Dynamic Programming (DP) is an optimization technique that:
1. **Breaks down complex problems** into simpler subproblems
2. **Stores solutions** to avoid recomputing them
3. **Builds up solutions** from smaller to larger problems

**Key Insight**: DP turns exponential O(2â¿) algorithms into polynomial O(n) or O(nÂ²)!
