## üîç Cheat Sheets

### Time Complexity Cheat Sheet
```
O(1)       - Hash table access
O(log n)   - Binary search
O(n)       - Linear scan
O(n log n) - Efficient sorting
O(n¬≤)      - Nested loops
O(2‚Åø)      - Recursive fibonacci (no memo)
```

### Python Built-in Complexity
- `list.append()` - O(1)
- `list.insert(0, x)` - O(n)
- `dict[key]` - O(1)
- `key in dict` - O(1)
- `list.sort()` - O(n log n)

### Common DP Patterns
1. **1D DP**: Fibonacci, House Robber
2. **2D DP**: Grid paths, LCS
3. **Knapsack**: 0/1, Unbounded
4. **Palindromes**: Subsequence, Partitioning
5. **Strings**: Edit Distance, Regex
6. **Trees**: Diameter, Path Sum
7. **Intervals**: MCM, Burst Balloons
