# Topic-Wise Problem Sets

This document contains 50+ curated problems to solidify your understanding of
each topic.

## 1. Arrays & Strings

- [ ] **Easy**: Two Sum, Valid Anagram, Reverse String
- [ ] **Medium**: Product of Array Except Self, Longest Consecutive Sequence,
      Group Anagrams
- [ ] **Hard**: First Missing Positive, Slidng Window Maximum

## 2. Linked Lists

- [ ] **Easy**: Reverse Linked List, Merge Two Sorted Lists
- [ ] **Medium**: Add Two Numbers, Copy List with Random Pointer, LRU Cache
- [ ] **Hard**: Reverse Nodes in k-Group, Merge k Sorted Lists

## 3. Stacks & Queues

- [ ] **Easy**: Valid Parentheses, Min Stack
- [ ] **Medium**: Daily Temperatures, Decode String, Evaluate Reverse Polish
      Notation
- [ ] **Hard**: Largest Rectangle in Histogram

## 4. Trees & Graphs

- [ ] **Easy**: Invert Binary Tree, Maximum Depth of Binary Tree
- [ ] **Medium**: Level Order Traversal, Validate BST, Number of Islands, Course
      Schedule
- [ ] **Hard**: Word Ladder, Binary Tree Maximum Path Sum

## 5. Dynamic Programming

- [ ] **Easy**: Climbing Stairs, House Robber
- [ ] **Medium**: Coin Change, Longest Increasing Subsequence, Word Break
- [ ] **Hard**: Edit Distance, Burst Balloons, Regular Expression Matching
