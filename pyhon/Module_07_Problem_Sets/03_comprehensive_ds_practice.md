# Comprehensive Data Structure Practice Set

A curated list of problems organized from **Basic** to **Advanced** for every
major data structure.

## 1. Arrays & Strings

- **Basic**:
  1. **Two Sum** (LeetCode 1): Introduction to HashMaps.
  2. **Best Time to Buy and Sell Stock** (LeetCode 121): One pass logic.
- **Intermediate**: 3. **Product of Array Except Self** (LeetCode 238):
  Prefix/Suffix logic. 4. **Longest Substring Without Repeating Characters**
  (LeetCode 3): Sliding Window.
- **Advanced**: 5. **Minimum Window Substring** (LeetCode 76): Hard Sliding
  Window. 6. **Trapping Rain Water** (LeetCode 42): Two Pointers /
  Pre-computation.

## 2. Linked Lists

- **Basic**:
  1. **Reverse Linked List** (LeetCode 206): Iterative & Recursive.
  2. **Merge Two Sorted Lists** (LeetCode 21): Basic pointer manipulation.
- **Intermediate**: 3. **Linked List Cycle II** (LeetCode 142): Cycle detection
  (Floyd's). 4. **Reorder List** (LeetCode 143): Middle + Reverse + Merge.
- **Advanced**: 5. **Merge k Sorted Lists** (LeetCode 23): Heap / Divide &
  Conquer. 6. **Reverse Nodes in k-Group** (LeetCode 25): Complex pointer
  management.

## 3. Stacks & Queues

- **Basic**:
  1. **Valid Parentheses** (LeetCode 20): Stack fundamentals.
  2. **Implement Queue using Stacks** (LeetCode 232): Understanding LIFO vs
     FIFO.
- **Intermediate**: 3. **Min Stack** (LeetCode 155): Auxiliary stack design. 4.
  **Daily Temperatures** (LeetCode 739): Monotonic Stack introduction.
- **Advanced**: 5. **Largest Rectangle in Histogram** (LeetCode 84): Monotonic
  Stack Application. 6. **Sliding Window Maximum** (LeetCode 239): Monotonic
  Deque.

## 4. Hash Tables

- **Basic**:
  1. **Contains Duplicate** (LeetCode 217): Set usage.
  2. **Majority Element** (LeetCode 169): Counting / Boyer-Moore.
- **Intermediate**: 3. **Group Anagrams** (LeetCode 49): Hashing complex keys.
  4. **Longest Consecutive Sequence** (LeetCode 128): HashSet sequencing.
- **Advanced**: 5. **Insert Delete GetRandom O(1)** (LeetCode 380): Systems
  design component. 6. **Subarray Sum Equals K** (LeetCode 560): Prefix Sum +
  HashMap.

## 5. Trees (Binary Trees & BST)

- **Basic**:
  1. **Maximum Depth of Binary Tree** (LeetCode 104): Simple Recursion.
  2. **Invert Binary Tree** (LeetCode 226): Tree traversal.
- **Intermediate**: 3. **Binary Tree Level Order Traversal** (LeetCode 102): BFS
  Queue. 4. **Validate Binary Search Tree** (LeetCode 98): BST constraints.
- **Advanced**: 5. **Serialize and Deserialize Binary Tree** (LeetCode 297):
  Full structure encoding. 6. **Binary Tree Maximum Path Sum** (LeetCode 124):
  Tree DP.

## 6. Heaps / Priority Queues

- **Basic**:
  1. **Last Stone Weight** (LeetCode 1046): Basic heap operations.
  2. **Kth Largest Element in a Stream** (LeetCode 703): Maintaining min-heap.
- **Intermediate**: 3. **Kth Largest Element in an Array** (LeetCode 215):
  QuickSelect vs Heap. 4. **Top K Frequent Elements** (LeetCode 347): Bucket
  Sort vs Heap.
- **Advanced**: 5. **Find Median from Data Stream** (LeetCode 295): Two Heaps
  balancing. 6. **Merge k Sorted Lists** (LeetCode 23): (Repeated for
  importance).

## 7. Graphs

- **Basic**:
  1. **Flood Fill** (LeetCode 733): Simple DFS/BFS.
  2. **Find the Town Judge** (LeetCode 997): In-degree/Out-degree.
- **Intermediate**: 3. **Number of Islands** (LeetCode 200): Connected
  components. 4. **Course Schedule** (LeetCode 207): Topological Sort / Cycle
  detection.
- **Advanced**: 5. **Word Ladder** (LeetCode 127): BFS Shortest Path. 6. **Alien
  Dictionary** (LeetCode 269): Topological Sort construction.

## 8. Tries

- **Basic/Intermediate**:
  1. **Implement Trie** (LeetCode 208): Standard implementation.
  2. **Design Add and Search Words** (LeetCode 211): Trie with wildcards.
- **Advanced**: 3. **Word Search II** (LeetCode 212): Trie + Backtracking
  optimization. 4. **Maximum XOR of Two Numbers in an Array** (LeetCode 421):
  Bitwise Trie.

## 9. Dynamic Programming (Bonus Depth)

- **Basic**:
  1. **Climbing Stairs** (LeetCode 70): Fibonacci.
  2. **House Robber** (LeetCode 198): 1D DP.
- **Intermediate**: 3. **Longest Common Subsequence** (LeetCode 1143): 2D Grid
  DP. 4. **Coin Change** (LeetCode 322): Unbounded Knapsack.
- **Advanced**: 5. **Edit Distance** (LeetCode 72): String manipulation DP. 6.
  **Burst Balloons** (LeetCode 312): Interval DP.
